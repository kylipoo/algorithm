Given N intervals [ai,bi] and one linear interval [s,t]，please select minimal intervals to cover the linear interval

Print out the least number of intervals, -1 if no solution.

input
s t for [s, t]
n intervals
ai bi each interval

output:
number of intervals or -1

constraints
1≤N≤10^5,
−10^9≤ai≤bi≤10^9,
−10^9≤s≤t≤10^9

Input sample:
1 5
3
-1 3
2 4
3 5
Output sample:
2
